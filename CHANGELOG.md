# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2024-05-23

### Added
- **Global Push-to-Talk**: Toggle recording from any application with `Alt+Space`.
- **Local Transcription**: Private, offline speech-to-text using Whisper WASM.
- **Microphone Selection**: Choose input device from Menu Bar or Settings.
- **Menu Bar Widget**: Control app, change modes, and quit from the system tray.
- **Auto-Paste**: Automatically insert text into the active application.
- **AI Enrichment**: Optional cleanup, formatting, and summarization via OpenAI.
- **Multi-language Support**: Support for 8+ languages in Local mode.
- **Settings Persistence**: All preferences saved and restored on launch.
